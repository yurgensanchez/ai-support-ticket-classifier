# Dataset Notes

## Source

`data/sample_tickets.csv` is a fictional dataset created for this portfolio project.

## Columns

| Column | Type | Description |
| --- | --- | --- |
| `text` | string | Customer support message. |
| `category` | string | Target label for classification. |

## Labels

- `sales`
- `technical_support`
- `billing`
- `complaint`
- `appointment`
- `other`

## Privacy

The dataset does not contain real customer data, names, emails, phone numbers, payment details, or private business information.

## Limitations

- The dataset is intentionally small.
- Messages are short and clean.
- It does not represent slang, multilingual messages, long conversations, or noisy real-world support data.
- Metrics from this dataset should be interpreted only as a local demo signal.
